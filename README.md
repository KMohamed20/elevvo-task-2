# Task 2: Contact Form

> **Level**: Level 1  
> **Requirements Met**: ✅ Full form fields | ✅ Validation | ✅ Responsive layout | ✅ Clean UI  

## 🔍 Description

A beautifully styled and fully functional contact form with client-side validation using **HTML, CSS, and JavaScript**. Designed to work seamlessly across all devices.

Form includes:
- Full Name
- Email Address (with format validation)
- Subject
- Message
- Submit Button

## 🖥️ Features

- ✅ **Real-time Validation**: Checks for required fields and valid email format.
- ✅ **Visual Feedback**: Input borders turn blue on focus; errors trigger alerts.
- ✅ **Mobile Responsive**: Stackable layout on small screens.
- ✅ **Modern Design**: Gradient background, card-based form, hover effects.
- ✅ **User Experience**: Clear labels, spacing, and intuitive flow.

## 🛠️ Technologies Used

| Technology | Purpose |
|----------|---------|
| HTML5    | Form structure and input types |
| CSS3     | Styling, flexbox, gradients, shadows |
| JavaScript | Validation logic and form submission handling |
